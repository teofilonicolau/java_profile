# Projeto PerfilApp Este é um projeto de demonstração que cria uma interface gráfica usando a biblioteca Swing do Java. Ele exibe informações pessoais e um link para um perfil no LinkedIn. ## Instruções de Uso 1. Clone este repositório para a sua máquina local. 2. Importe o projeto no Eclipse. 3. Execute o projeto para ver a interface gráfica. ## Requisitos - Eclipse IDE - Java Development Kit (JDK) - Conexão à internet (para acessar o LinkedIn) ## Recursos Utilizados - Biblioteca Swing para a interface gráfica. - Linguagem de programação Java. - URL do perfil do LinkedIn: [Teófilo Nicolau](https://www.linkedin.com/in/teofilo-nicolau/) ## Contribuição Contribuições são bem-vindas! Se você tiver melhorias ou correções, sinta-se à vontade para fazer um fork deste repositório e enviar um pull request. ## Licença Este projeto é distribuído sob a licença [MIT](LICENSE).
